CSS Box Model

CSS box model is majorily divided into four components:-

  1. Margin
  2. Border
  3. Padding
  4. Content

Margin

The empty space separating the element from its neighboring items is known as margin. It is the outermost layer of the CSS box model. Its dimensions are given using margin-box width and margin-box height. Margin can be defined by the following properties: margin-left, margin-right, margin-top, margin-bottom or the shorthand margin property.

Border

The property that lets us add and style a line around the content padded area is known as border. The thickness of the border is given by border-width property, color by border-color property and style by border-style property or we can also use a shorthand border property to define all three. Border-style contains values like solid, dotted, dashed, double, groove, ridge, and none.

Padding

The space between the border and content of an element is known as padding. If padding is applied in a proper way it helps to make the content more visible and readable. Good padding also enhances the layout of the website. Padding can be added to a particular element using the following properties: padding-top, padding-bottom, padding-left, padding-right, or the shorthand property padding.

Content

The innermost rectangle is known as the content area which contains text or other visual elements which developer wants to show to the user. It is defined using content width and content height or it can also be given properties like min-width, max-width, min-height, and max-height to display it according to the user's display size.
