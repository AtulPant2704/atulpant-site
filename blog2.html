Some Git commands which are often used to make a developer's job easy are given below:-

  1. git init

    2. This command helps in creating a new empty repository or to reinitialize the existing one if their is some error in the existing one. It will create a .git directory in the root of the project.

  2. git clone

    3. This command is used for downloading the latest version of a remote project and copying it to the selected location on the local machine.

  3. git pull

    4. This command fetches all the changes from the remote repository and than merges them with the code in the current local branch.

  4. git push

    5. This command helps in pushing the locally committed changes to the remote branch.

  5. git commit

    6. This command commits or saves the changes that you have done locally on your machine. It helps to trackdown the development of an application if the developer wants to come back to the previous code.

  6. git branch

    7. Using git branch will list all the branches of the repository or it can also be used to create or delete a branch.

  7. git merge

    8. This command merges the code from one branch to the other branch, often from remote to the main branch.

  8. git checkout

    9. This command is used to switch between different branches and start working in a new one.
